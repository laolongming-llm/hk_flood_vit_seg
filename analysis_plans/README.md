# 分析规划文档目录说明

| 项目 | 内容 |
|---|---|
| 概述 | 存放项目过程中阶段性分析、方案设计、决策记录与复盘文档。 |
| 时间 | 2026-04-08 |

## 使用约定
- 文档统一存放在 `analysis_plans/`。
- 文件命名建议：`YYYY-MM-DD_主题关键词.md`。
- 每个 Markdown 开头统一包含“概述 + 时间”信息表，便于检索与时间线复盘。

## 推荐开头模板
```markdown
# 标题

| 项目 | 内容 |
|---|---|
| 概述 | 一句话说明本文目标与结论。 |
| 时间 | YYYY-MM-DD |
```
